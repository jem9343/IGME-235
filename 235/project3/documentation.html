Brainstorming

When searcing for APIs, I wasn't sure what I wanted to do for this project. I never considered myself good at working with data, so I wanted to find a way to get data that I found interesting.

I originally wanted my project to be an recipe finder based on ingredients, but all of the APIs that had that capability needed a key. I decided to search around on the Jikan API, and I was excited to find that there was a manga search option!

Proposal

For my project 2, I am planning on creating a manga database tool with a search function utilizing the Jikan API. The main function will be a general search bar to get a manga by title, as well as advanced search features including the type of manga (like "oneshot", "doujin", and "manhwa"), filtering for a minimum/maximum score, as well as current status ("publishing" "complete" "hiatus" "discontinued" "upcoming").
